# 🔳 Python QR Code Generator

A simple Python project that generates a QR code from any URL and saves it as an image on the Desktop.

## 🚀 Features

* Generate QR codes from any URL
* Save the QR code as a PNG image
* Simple and beginner-friendly Python code
* Uses the `qrcode` and `Pillow` libraries

QRCODE/

## 📦 Installation

### 1. Clone the repository

```bash
git clone (https://github.com/anujkush2431393/python-qr-code-generator.git)
cd QRCODE
```

### 2. Create a virtual environment

```bash
python -m venv .venv
```

### 3. Activate the virtual environment

**Windows (PowerShell)**

```powershell
.\.venv\Scripts\Activate.ps1
```

### 4. Install dependencies

```bash
pip install qrcode[pil]
```

---

## ▶️ Run the Project

```bash
python main.py
```

The program will ask for a URL.

Example:

```text
Enter the URL: https://github.com/anujkush2431393
```

After entering the URL, a QR code image will be generated and saved to:

```text
C:\Users\Hp\OneDrive\Desktop\qrcode.png
```
